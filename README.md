# Starter Project

A minimal static starter project with:

- `index.html`
- `styles.css`
- `script.js`

## Run locally

1. Open `index.html` in your browser.
2. Or use a local server, for example:

```bash
echo "Serving from current directory"
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
